# Promoted Project-Imp Lessons

Local lessons in `../local/` are machine state and are ignored. When a lesson
has prevented a repeat failure or led to a durable test/probe, promote it here
or into the owning imp prompt/registry entry.

Promotion destinations:

- Permanent imp prompt: repeated command or workflow failure.
- `AGENTS.md`: cross-cutting repo rule.
- Focused test/probe: user-visible regression.
- Owning docs or `.notes`: durable product/domain assumption.
