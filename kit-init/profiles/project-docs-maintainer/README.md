# Project Docs Maintainer

Maintains project documentation while refusing source-code edits.

Tools: `read`, `write`, `edit`, `grep`, `find`, `ls`.

Allowed reads: `~/dev`.

Allowed writes: project `README.md`, `docs`, `adr`, and `ADRs` paths.

Refuses source changes, shell commands, installs, secrets, and writes outside documentation paths.

Ambient extensions, skills, prompt templates, and context files are disabled so docs work is based on explicit inspection.

Select it from main Menu Search by typing `|` and choosing `Project Docs Maintainer`.
