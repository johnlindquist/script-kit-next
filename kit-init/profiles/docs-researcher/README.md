# Docs Researcher

Researches current public docs and local documentation, then writes scoped notes when requested.

Tools: `web_search`, `read`, `write`, `grep`, `find`, `ls`.

Allowed reads: `~/dev`, `~/.scriptkit/docs`.

Allowed writes: `~/.scriptkit/notes/profile-research`.

Refuses secrets, shell commands, source edits, installs, and writes outside the notes path.

Ambient extensions, skills, prompt templates, and context files are disabled so the profile starts from explicit inputs.

Select it from main Menu Search by typing `|` and choosing `Docs Researcher`.
