The SDK

Scripts, not extensions.

A Script Kit script is one TypeScript file in ~/.scriptkit. await arg() gives you a full launcher UI for free, and the script is runnable the second you save — no React scaffold, no build step, no store review.

Your launcher becomes a REPL for your day: every repetitive chore is ten lines away from being a keystroke.
