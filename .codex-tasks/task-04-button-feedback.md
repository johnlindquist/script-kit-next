# Task: Implement Button Click Feedback System

Create or enhance a system for providing visual feedback when buttons are clicked.

1. Search for existing HUD/toast/notification systems in the codebase
2. Find buttons that lack visual feedback on click
3. Implement feedback for buttons - options include:
   - Brief flash/highlight on the button
   - HUD toast notification
   - Status message display

Look at:
- `src/notifications/` or similar
- Any existing toast/HUD implementations
- How other actions provide feedback

Ensure every button click results in some visible response to the user.
