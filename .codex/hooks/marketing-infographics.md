# Marketing Infographics Stop Run

This brief tells the Codex Stop hook how to continue Script Kit GPUI marketing image generation.

## Brand Anchors

Script Kit GPUI is fast, focused, and minimal. It is a keyboard-first native macOS launcher for power developers who want precise automation without extra chrome.

- Primary visual signal: `assets/logo.svg` and `assets/icon.png`.
- Brand personality: fast, focused, minimal.
- Signature accent: gold `#fbbf24`.
- Core surface: dark native macOS vibrancy with crisp command-palette rows.
- Product vocabulary: Run, Actions, AI, Proof, scripts, shortcuts, stdin JSON, receipts.

## Image Run Contract

Each continuation pass should create one new marketing infographic with `$imagegen`.

Do the repo checks and any required reading before the image call. The image generation call should be the final action of the pass, because the built-in image tool response must not be followed by commentary.

Use the project theme and logo cues as much as practical, but vary the visual treatment broadly across passes. Acceptable styles include realistic photography, product renders, editorial posters, historical illustration, wild technical diagrams, packaging concepts, zine collage, mission-control dashboards, and deliberately ridiculous but still legible campaign imagery.

Keep image text short and exact. Good phrases: `Script Kit`, `Fast. Focused. Minimal.`, `Run`, `Actions`, `AI`, `Proof`, `Automate everything`.

Avoid watermarks, fake unrelated brand names, long paragraphs, unreadable tiny labels, and screenshots that imply product UI states that do not exist.
