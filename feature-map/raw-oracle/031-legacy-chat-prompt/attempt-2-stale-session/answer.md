Operator-grade feature atlas: 031 Legacy chat() Prompt 1. Capability summary The legacy chat() prompt is Script Kit GPUI’s older conversational prompt surface. It is exposed through the TypeScript SDK in scripts/kit-sdk.ts and rendered by the Rust/GPUI ChatPrompt entity. It is a Raycast-style chat UI where scripts can show initial messages, accept user submissions, render assistant/user messages, stream responses, show errors, clear history, and optionally delegate AI generation to the app’s built-in provider layer. Primary users: Script authors using SDK chat() for conversational scripts. Runtime/app code routing JSON protocol messages into AppView::ChatPrompt. Human users typing messages, using actions, copying responses, clearing chats, selecting models, attaching images, or handing off to the full AI window. Agents/test harnesses driving the surface through stdin JSON protocol, setInput, simulated keys, and state snapshots. This feature is distinct from ACP Chat SDK APIs (ai*). ACP Chat has its own API surface and AppView::AcpChatView; this legacy surface uses SDK chat(), protocol messages like type: "chat", type: "chatSubmit", type: "chatMessage", and Rust AppView::ChatPrompt.
